1. git으로 관리할 경로까지 이동

2. 초기화
git init

3. 브랜치 이름 기본값 변경하기
git config --global init.defaultBranch main

4. git 상태확인
git status

5. git 으로 관리하는 파일로 추가
git add .

6. commit(해당 시점을 저장)
git commit -m "커밋 메세지"
(ex) git commit -m "first"

7. commit 이전에 계정정보 등록(--global 옵션은 해당컴퓨터 전체에 적용)
git config --global user.email "you@example.com"
git config --global user.name "Your Name"

8. commit 정보 확인
git log

로컬 컴퓨터(코드있음)와 github 사이트(비어있음) 연결
1. remote 정보 확인
git remote
git remote -v

2. remote 추가
git remote add 리모트 이름 github리파지토리주소
git remote add origin https://github.com/swj3140/project5_work1.git

3. remote 삭제
git remote rm 리모트이름
git remote rm origin

4. github 데이터 넣기
git push -u origin main
git push
